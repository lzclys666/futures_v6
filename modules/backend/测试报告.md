## 二、问题与修复

### 问题1: IC矩阵返回全0

**原因**: data_access.py 只检查 _shared 目录，未检查 shared 目录

**修复**: 增加 shared 目录检查，静默处理缺失文件

### 问题2: factorBreakdown 为空

**原因**: load_factor_value 只检查 value 列，实际列名不匹配

**修复**: 检查多个可能的列名（value / au_ag_ratio_corrected / usd_cny 等）

### 问题3: 前端端口对接

**结论**: 前端需同时对接 8000（VNPy交易）和 8001（因子分析）
