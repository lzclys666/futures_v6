## 三、根因分析

### 3.1 icValue 全部为 0.0

**所有 13 个因子的 icValue 均为 0.0**，这是核心问题。根因是：

- IC（Information Coefficient）需要 ≥30 个交易日的因子收益率数据才能计算
- AG 品种的 PIT 数据积累不足 30 天（系统上线 ~4/21，至今仅 ~10 天）
- **IC = 0 → 权重退化为等权（0.0769 = 1/13）**，但 `normalizedScore` 也全为 0

### 3.2 normalizedScore = 0 的直接原因

`rawValue` 有值但 `normalizedScore = 0` 意味着**标准化环节出问题**：

1. **MAD 标准化需要历史窗口**（通常 20~60 日滚动窗口）
2. PIT 数据积累不足 → 无足够历史数据做标准化 → 输出 0
3. **AG_FUT_CLOSE 是唯一非零**：可能是因为日线价格数据有独立来源（AKShare 直接采集），不依赖 PIT 数据

### 3.3 结论

- **非 Bug**：这是数据积累期的预期行为
- **预计恢复节点**：6/9~6/16（P1-3 结论），届时多数因子达到 30 交易日门槛
- **AG_FUT_CLOSE 例外**：该因子使用独立价格序列，不受 PIT 数据积累限制

## 四、建议行动

3. **6/9 复查**：到因子复活节点时重新验证 AG 品种 IC 和 normalizedScore 恢复情况
