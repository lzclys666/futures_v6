## 1. 方法论说明

### 1.1 验证框架
- **IC（Information Coefficient）**：60日滚动窗口内因子与前瞻收益率的 Pearson 相关系数
- **IR（Information Ratio）**：IC均值 / IC标准差
- **前瞻期**：默认5日（fwd=5），同时测试1/10/20日
- **显著性检验**：t检验（H0: IC=0），FDR校正后 p<0.05
- **PIT合规**：所有分析使用历史数据，无未来函数

### 1.2 评级标准

| 评级 | IC均值 | IR | t统计量 | 说明 |
|------|--------|-----|---------|------|
| 🟢 EXCELLENT | >0.08 | >1.0 | >6.0 | 顶级信号，立即上线 |
| 🟡 GOOD | >0.05 | >0.5 | >3.0 | 强信号，上线推荐 |
| 🟠 PASS | >0.02 | >0.3 | >2.0 | 中等信号，可上线 |
| ⚫ WEAK | >0 | >0.1 | >1.0 | 弱信号，观察 |
| ❌ INVALID | ≤0 | — | — | 无预测能力 |

## 5. 因子组合策略框架

基于Phase 1验证，建议采用三层信号组合：

### 第一层：核心趋势信号（P1，上线优先级）
```
IF WTI_diff > 0 AND LME_Cu_SpreadDiff > 0:
    → 工业品多头信号（CU, NI, AG偏多）
IF USD_CNY_diff > 0:
    → 全品类空头信号（CU, AU, AG偏空）
```

### 第二层：相对价值信号（P2，辅助验证）
```
IF 金银比 > 历史75分位:
    → 白银相对黄金被高估 → 做空白银/做多黄金
IF CN_10Y2Y_spread扩大:
    → 宏观衰退预期 → 黄金多头
```

### 第三层：反转信号（P3，观察）
```
IF CN10Y大幅上升（>10bp/日）:
    → 利率压制黄金 → 偏空（但信号频率低）
```
