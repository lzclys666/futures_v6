## 输出要求

采集数据请按以下格式存入本地数据库（字段名严格遵守）：

```
Table: sa_raw_data
- symbol        VARCHAR(10)    -- 'SA' 或 'FG' 等
- date          DATE           -- 数据日期
- field_name    VARCHAR(64)    -- 因子英文名（如 total_inventory, operating_rate）
- value         DECIMAL(18,4)  -- 数值
- unit          VARCHAR(16)    -- 单位（万吨、%、元/吨等）
- source        VARCHAR(64)    -- 数据来源
- fetch_time    DATETIME       -- 采集时间
- is_final      BOOLEAN        -- 是否为终稿（False=初值待修正）
```
