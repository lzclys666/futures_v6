## 1. IC 热力图 API

### 端点
`GET /api/ic/heatmap`

### 请求参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| symbols | string[] | 否 | 品种列表，默认 ['RB','HC','I','J'] |
| lookback | int | 否 | 回看天数，默认 60 |

### 响应格式
```json
{
  "factors": ["库存变化", "基差率", "现货升贴水", ...],
  "symbols": ["RB", "HC", "I", "J"],
  "icMatrix": [[0.15, -0.08, 0.22, ...], ...],
  "lookbackPeriod": 60,
  "updatedAt": "2026-04-26T15:30:00Z"
}
```

### ✅ 已确认

| 项目 | 确认内容 |
|------|----------|
| IC 计算方法 | **Spearman** |
| 品种筛选 | ✅ 支持多选 |
| 更新频率 | **日终** |
| 热力图颜色 | **红=负IC，绿=正IC** |
| 预计完成 | **2026年05月05日** |

## 2. 信号系统 API

### 端点
`GET /api/signal/{symbol}`

### 响应格式
```json
{
  "symbol": "RB",
  "compositeScore": 65.5,
  "signalStrength": "BUY",
  "confidence": 78.5,
  "factorBreakdown": [
    {
      "factorName": "库存变化",
      "weight": 0.25,
      "ic": 0.15,
      "contribution": 12.5
    }
  ],
  "regime": "TRENDING",
  "timestamp": "2026-04-26T15:30:00Z"
}
```

### ✅ 已确认

| 项目 | 确认内容 |
|------|----------|
| 方案 | **方案 B**（新建端点） |
| 信号聚合 | **IC加权求和** |
| 计算频率 | **日终** |
| UI展示 | **仪表盘（当前）+ 列表（历史）** |
| 预计完成 | **2026年05月12日** |

### 信号强度定义
| 分值范围 | 信号强度 |
|----------|----------|
| 0-20 | STRONG_SELL |
| 20-40 | SELL |
| 40-60 | NEUTRAL |
| 60-80 | BUY |
| 80-100 | STRONG_BUY |

### Regime 定义
| 状态 | 说明 |
|------|------|
| TRENDING | 趋势市场 |
| RANGING | 震荡市场 |
| HIGH_VOLATILITY | 高波动 |
| LOW_LIQUIDITY | 低流动性 |
