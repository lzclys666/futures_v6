> Rolling IC（60日窗口）是标准IC，衡量因子在滚动窗口内的**稳定**预测能力。数值代表跨多段窗口的平均IC。

### 2.2 核心发现

#### 🔥 发现 1：SGE 黄金现货 → 沪金期货，存在 Same-Day 超强领先

**滚动 IC**: IC=+0.017（弱，不稳定）
**1日衰减 IC**: IC=+0.682（IR=5.07，极强）

这意味着在**某些时期**，SGE黄金现货当日上涨 → 沪金期货当日跟涨，且信号极强（IC=0.68）。但该关系**不稳定**——60日滚动平均后IC降至0.017，说明信号强度在不同市场环境下差异极大。

**经济学逻辑**：上海黄金交易所（SGE）是全球最大的实物黄金市场之一（仅次于伦敦），交易时间与SHFE黄金期货高度重叠（亚洲时段）。SGE现货价领先SHFE期货，但信号强度受国际金价走势主导。

**策略含义**：
- 信号方向正确（SGE↑ → 沪金↑），但**不是稳定Alpha**
- 适合在信号强度高（IC > 0.1）时使用，不适合作为单一策略

---

#### 🔥 发现 2：金银比（AUD/AGD）是沪银的极佳反向指标

**Rolling IC（AG目标）**: IC=**-0.1605**，IR=**-1.451**，胜率=**5.4%**

> 这是目前所有Phase 1因子中，**IR绝对值最高**的信号（IR=1.451）。

**含义**：金银比上升 → 沪银期货未来收益率为**负**（胜率仅5.4%，几乎必然下跌）。

**金银比经济学逻辑**：
- 金银比衡量黄金相对白银的价值（黄金/白银价格）
- 比值上升：黄金相对贵，白银相对便宜 → 反映经济扩张期（白银工业属性强，需求疲弱）
- 经济下行/避险情绪：金银比飙升 → 白银跌幅 > 黄金 → 沪银做空信号
- 胜率5.4%不是"错误"，而是意味着金银比上升时，**超过94%的情况下沪银确实下跌**

**金银比 → 沪金（反向较弱）**：
- IC=-0.074，胜率27.2%
- 金银比对沪银的影响远大于沪金（IR=-1.451 vs -0.674）
- 这是因为白银的工业属性更强，金银比反映的宏观工业周期对白银影响更大

---

#### 🔥 发现 3：WTI 原油 → 沪金（PASS，IC=+0.036）

- 信号方向：WTI上涨 → 沪金偏强
- 强度：IC=+0.036，胜率61.4%
- 衰减：1日IC=+0.011，5日IC=+0.007，10日IC=+0.034——信号**持续稳定**，不反转
- 评级：🟠 PASS

**经济学解释**：能源价格 → 通胀预期 → 黄金作为通胀对冲工具受益。但相关性本身偏弱（IC=0.036 < IC_GOOD=0.05），作为辅助信号，不适合单独使用。

---

#### 🔥 发现 4：中债10年国债收益率 → 沪金（PASS，IC=-0.048）

- 信号方向：债券收益率上升 → 沪金偏弱
- 强度：IC=-0.048，胜率29.3%
- 衰减：1日IC=-0.036，5日IC=+0.007，10日IC=+0.021——信号存在但不稳定
- ⚠️ 数据仅246条（1年），统计可信度有限

**经济学逻辑**：债券收益率=无风险利率=机会成本。利率上升 → 黄金持有成本上升 → 金价承压。逻辑正确，但数据量不足。

### 3.2 核心发现

#### 🟢 发现 5：金银比 → 沪银（EXCELLENT，IC=-0.1605）

这是本次贵金属验证的**最强信号**：
- IR=1.451（信息比率极高）
- t=-49（统计显著性极高）
- 胜率5.4%（意味着金银比上升时，沪银几乎必然下跌）

**交易策略**：
```
金银比 > 80（历史高位）：
  → 做空沪银期货
  → 持有期：5~10日
  → 止损：金银比回落至75以下

金银比 < 60（历史低位）：
  → 做多沪银期货
  → 持有期：5~10日
```

**注意**：金银比是一个**宏观叙事指标**，不是高频因子。它的信号频率低（可能数月一次），但胜率和幅度极高，是优质的**趋势信号**而非**均值回归信号**。

---

#### 🟡 发现 6：WTI 原油 → 沪银（GOOD，IC=+0.050）

- 信号强度高于沪金（IC=0.050 vs IC=0.036）
- 胜率62.9%
- 衰减模式：1日+0.014，5日-0.027，10日-0.018
- 白银的工业属性更强，对能源价格更敏感

---

#### 🟡 发现 7：黄金价格 → 白银收益率（PASS，IC=+0.034）

- SGE黄金现货收益率 → 沪银期货收益率
- IC=+0.034，胜率72.8%
- 方向正确，但强度一般
- 可作为**贵金属板块动量**的辅助信号

- **宏观因子**（WTI/USD）：对所有品种均有辅助信号，价值在于与其他因子的组合
