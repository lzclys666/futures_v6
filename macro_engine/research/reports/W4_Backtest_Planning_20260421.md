## 三、回测框架设计

### 3.1 框架选择：**自研 pandas/numpy**

| 方案 | 优点 | 缺点 | 决策 |
|------|------|------|------|
| Backtrader | 成熟、事件驱动 | 无中文文档，不支持多品种因子组合 | ❌ 不选 |
| Zipline | 量化事实标准 | 安装复杂，依赖多 | ❌ 不选 |
| Vectorbt | 功能强大 | 未安装，需重新配置环境 | ❌ 暂缓 |
| **自研 pandas/numpy** | **灵活、可控、代码透明** | **需自己实现** | **✅ 选用** |

**自研框架核心逻辑**：
```python
# 伪代码
for date in trading_calendar:
    # 1. 获取当日因子值（PIT快照）
    factors = get_factor_snapshot(date)
    
    # 2. 计算因子信号（z-score / 排序）
    signals = compute_signals(factors)
    
    # 3. 仓位分配（IC加权 / 等权 / 风险平价）
    positions = allocate_positions(signals, method='ic_weight')
    
    # 4. 计算当日盈亏
    pnl = positions * forward_return(date+hold)
    
    # 5. 更新组合
    portfolio.update(date, pnl)
```

### 3.2 因子信号计算

| 步骤 | 方法 | 说明 |
|------|------|------|
| 因子对齐 | 以交易日对齐，去除重复日期 | 共同交易日≥30天才纳入组合 |
| 因子标准化 | Z-score（rolling mean/std，60日窗口） | 剔除量纲差异 |
| 信号方向 | 根据 IC 方向确定多空 | IC>0 → 做多；IC<0 → 做空 |
| 因子合成 | IC加权（按 IR 比例）或等权 | 初期测两种方案 |

### 3.3 仓位管理

| 方案 | 说明 | 适用场景 |
|------|------|----------|
| 等权 | 每个因子权重相同 | 初期基准组 |
| IC加权 | 按 IC 比例分配权重 | 核心组合推荐 |
| 风险平价 | 每因子对组合风险贡献相同 | 稳健组合 |

## 五、评估指标

### 5.1 核心指标（必须输出）

| 指标 | 计算方法 | 阈值/说明 |
|------|----------|----------|
| **年化收益率** | (1+累计收益)^(252/交易日)-1 | 目标 > 15% |
| **夏普比率 (Sharpe)** | 年化收益/年化波动率 | 目标 > 1.0 |
| **最大回撤 (Max Drawdown)** | 历史最大回撤 | 目标 < 20% |
| **胜率 (Win Rate)** | 正收益交易次数/总交易次数 | 目标 > 52% |
| **盈亏比 (Profit Factor)** | 总盈利/总亏损 | 目标 > 1.3 |
| **卡玛比率 (Calmar)** | 年化收益/最大回撤 | 目标 > 1.0 |
| **Sortino比率** | 年化收益/下行波动率 | 目标 > 0.8 |
| **交易次数/年** | 252/持有期 | 参考指标 |

### 5.2 因子专项指标

| 指标 | 说明 |
|------|------|
| IC均值 | 因子预测能力（目标 > 0.02） |
| IR | 因子稳定性（目标 > 0.5） |
| 因子换手率 | 月度调仓换手（目标 < 30%） |
| 因子衰减半衰期 | 信号有效持续时间（目标 > 3天） |
| 因子拥挤度 | 持仓集中度风险（目标 < 60） |

### 5.3 对比基准

| 基准 | 说明 |
|------|------|
| 买入持有 (Buy & Hold) | 各品种主力合约买入持有 |
| 等权基准 | 所有因子等权组合 |
| 无风险利率 | 1年期国债收益率（约2%） |
