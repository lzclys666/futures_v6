### 4.2 数据流对接

```
因子升级输出 → 系统架构输入
═══════════════════════════════════════════════════════════════

HMM Regime状态 ──→ 宏观熔断机制
  └─ REGIME_STRONG   ──→ 正常交易
  └─ REGIME_NORMAL   ──→ 正常交易
  └─ REGIME_WEAK     ──→ 触发熔断审查

信号评分(0~100) ──→ 策略信号强度
  └─ 评分>80 ──→ 满仓信号
  └─ 评分50~80 ──→ 半仓信号
  └─ 评分<50 ──→ 观望/空仓

失效预警 ──→ 风控规则
  └─ IC持续低迷 ──→ 暂停该因子信号
  └─ 拥挤度>85 ──→ 降低该因子权重

IC热力图 ──→ 前端展示
  └─ 品种×因子矩阵 ──→ 交易监控面板嵌入
```
