## 因子配置

- `AU_CFTC_NC` — 黑币金钻非商业净多头
- `AU_DXY` — DXY = 美联储贸易加权美元指数，反映美国出口竞争力与全球资本流向。日度数据。
- `AU_FED_DOT` — FOMC委员对2026年末联邦基金利率预测中位数，反映美联储前瞻指引。季度数据，每次FOMC会议更新。
- `AU_FED_RATE` — 美联储联邦基金目标利率（Federal Funds Rate Target），反映美联储货币政策立场。
- `AU_GOLD_RESERVE_CB` — 中央银行黄金储备(吨)
- `AU_SHFE_RANK` — SHFE AU前20名会员净持仓（多空轧差手数），反映机构在沪金上的净投机/套保方向。日度数据。
- `AU_SPD_AUAG` — 金银比 = SGE黄金现货午市价(元/克) / SGE白银现货午市价(元/千克÷1000)。金银比↑表示黄金相对白银偏强（多金空银机会）；金银比↓表示白银相对黄金偏强（多银空金机会）。
- `AU_SPD_BASIS` — 黑币期现基差=SGE玻净-SHFE期货
- `AU_SPD_GLD` — SPDR Gold Shares (GLD) 持仓量(吨)，全球最大实物黄金ETF，反映机构黄金配置动向。日度数据。
- `AU_SPOT_SGE` — SGE金条玻净价标准价(元盎司克)
- `AU_US_10Y_YIELD` — 美国10年期国债收益率（名义），作为TIPS实际收益率的代理指标。AKShare无TIPS专属函数，使用bond_zh_us_rate()美国国债收益率10年字段作为替代。
