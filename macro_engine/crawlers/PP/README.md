## 脚本说明

### 入口脚本
| 脚本 | 说明 |
|------|------|
| PP_run_all.py | 编排器，依次调用 PP_聚丙烯期货收盘价.py 和 PP_聚丙烯期货持仓量.py |

### 数据采集脚本（2个有效）
| 脚本 | 说明 |
|------|------|
| PP_聚丙烯期货收盘价.py | AKShare L1 → 新浪期货PP主连收盘价 |
| PP_聚丙烯期货持仓量.py | AKShare L1 → 新浪期货PP主连持仓量 |

### Stub 占位脚本（10个）
所有 Stub 脚本均写入 `source_confidence=0.5` 的占位记录，等待付费订阅或接口验证后替换。

---

## 运行方式

```bash
# 批量运行
python PP_run_all.py --auto

# 单独运行某个因子
python PP_聚丙烯期货收盘价.py --auto
```
