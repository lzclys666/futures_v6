# 期货智能交易系统 V6 — 用户手册

## 快速开始

1. 登录系统（默认 admin/admin）
2. 检查 VNpy 连接状态（顶栏指示灯）
3. 查看宏观看板获取交易信号
4. 在交易面板下单

## 交易流程

1. **查看信号**：宏观看板 → 选择品种 → 查看复合评分
2. **风控预检**：交易面板 → 输入订单 → 系统自动预检
3. **下单**：预检通过 → 点击「买入/卖出」
4. **监控**：持仓看板实时更新

## 风控规则

系统内置 11 条风控规则，分三层：

- **L1 宏观层**：宏观评分熔断、方向冲突
- **L2 账户层**：仓位、亏损、保证金、波动率
- **L3 执行层**：交易时间、订单频率、权限检查

## 常见问题

**Q: 为什么下单按钮是灰色的？**
A: 风控预检未通过，查看具体阻断规则。

**Q: 如何切换 Mock/真实数据？**
A: 修改 `.env.local` 中 `VITE_USE_MOCK`。

**Q: 深色模式在哪里？**
A: 顶栏右侧月亮图标（Phase 5 完整实现）。
