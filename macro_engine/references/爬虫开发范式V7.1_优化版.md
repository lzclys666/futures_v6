## 二、数据源降级模型（核心）

### 2.1 四级降级结构

```
每个因子的获取逻辑：
┌─────────────────────────────────────────────────────────┐
│ 源1（主力）→ 源2（备用）→ 源3（DB兜底）→ 源4（手动）│
│   ↓ 失败        ↓ 失败        ↓ 失败/过期      ↓ 非auto模式  │
│ confidence=1.0   confidence=0.9  confidence=0.6  confidence=0.5 │
└─────────────────────────────────────────────────────────┘
```

### 2.2 源选择优先级（成本分层策略）

| 优先级 | 数据源类型 | 特点 | 示例 |
|--------|-----------|------|------|
| **L1 免费权威** | 官方机构、交易所、开源接口 | 最权威、免费、稳定 | AKShare、Tushare、FRED、国家统计局、交易所官网 |
| **L2 免费聚合** | 金融平台公开数据 | 免费、覆盖广、稳定性稍弱 | 东方财富、新浪财经、同花顺公开页面 |
| **L3 付费源** | 商业终端、行业资讯站 | 付费、深度数据 | Mysteel(年费)、汾渭(年费)、普氏(付费) |

**选源规则：**
- 源1：从L1池选最可靠的可用源
- 源2：从L1池第二选择，或L2池最佳选择
- 如果L1/L2均无可用源，L3付费源可作为源1（需标注付费平台）
- 每层独立 try-except，上一层成功即返回，不触发下一层

### 2.3 Source 3 分场景处理（Token优化）

| 因子类型 | Source 3策略 | 理由 |
|----------|------------|------|
| **日频免费因子** | ❌ 去掉 | 两源都挂概率极低，真挂了手动输入更干净 |
| **周频免费因子** | ❌ 去掉 | 同上 |
| **月频免费因子** | ✅ 保留 | 月末几天用上月值是正常的 |
| **纯付费因子** | ✅ 保留 | DB回补是主力路径 |

## 五、Token节约规范

### 5.1 爬网页输出规范（最重要）

```python
# ❌ 禁止：打印原始HTML
print(f"网页内容: {html[:1000]}...")

# ✅ 正确：只输出解析结果
print(f"[源X] 网页解析成功: {value}")

# ✅ 错误时也只输出简要信息
except Exception as e:
    print(f"[源X] 解析失败: {e}")  # 不打印html
```

### 5.2 运行输出分级

```python
# 正常模式：只输出关键结果
print(f"[源1] 成功: {value}")

if "--verbose" in sys.argv:
```

|------|------|----------|
| 期货实时行情 | `futures_zh_spot()` | 返回中文列名 |

```python
# ❌ 原来：逐个请求
for symbol in ["RU", "BR", "AG"]:

df = ak.get_receipt(vars_list=["RU", "BR", "AG"])  # 1次请求
```

## 七、开发流程

### 7.1 实测驱动开发

```
1. 创建 {SYMBOL}_debug_sources.py 探查接口
2. 运行验证，记录返回格式、列名、编码
3. 输出 {SYMBOL}_source_validation.md
4. 复制 _template.py，修改配置区
5. 实现 fetch_data() 函数体
6. 测试运行，确认数据入库
7. 清理临时调试文件
```

### 7.2 联动检查清单

每完成一个品种，逐项确认：

| 检查项 | 验证方法 |
|--------|----------|
| `crawlers/update_all.py` 的 SYMBOLS 列表 | 品种已加入 |
| `config/factors/{SYMBOL}/` 目录 | YAML文件名与FACTOR_CODE一致 |
| `config/instruments/{SYMBOL}.yaml` | factors列表包含所有因子 |
| `{SYMBOL}_run_all.py` | scripts列表包含所有脚本 |
| 数据库写入 | `SELECT DISTINCT factor_code FROM pit_factor_observations WHERE symbol='XX'` |
| 临时文件清理 | `del *_debug.py *_test.py 2>nul` |
