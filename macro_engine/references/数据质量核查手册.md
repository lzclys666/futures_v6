## 三、深度检查命令

### 查看某品种近3天因子数据

```bash
python -c "
import sqlite3, sys
db = 'd:/futures_macro_engine/pit_data.db'
conn = sqlite3.connect(db)
cur = conn.cursor()
species = sys.argv[1] if len(sys.argv) > 1 else 'BR'
cutoff = (datetime.date.today() - datetime.timedelta(days=3)).isoformat()
cur.execute('''
    SELECT factor_code, obs_date, raw_value
    FROM pit_factor_observations
    WHERE factor_code LIKE ? AND obs_date >= ?
    ORDER BY obs_date DESC, factor_code
''', (species + '%', cutoff))
for r in cur.fetchall():
    print(r)
" BR
```

### 快速获取数据库健康评分

```python
# 直接查询核心指标（Python交互式）
import sqlite3
from datetime import date, timedelta

db = 'd:/futures_macro_engine/pit_data.db'
conn = sqlite3.connect(db)
cur = conn.cursor()

cutoff7 = (date.today() - timedelta(days=7)).isoformat()
cutoff30 = (date.today() - timedelta(days=30)).isoformat()

total = cur.execute('SELECT COUNT(DISTINCT factor_code) FROM pit_factor_observations').fetchone()[0]
recent7 = cur.execute('SELECT COUNT(DISTINCT factor_code) FROM pit_factor_observations WHERE obs_date >= ?', (cutoff7,)).fetchone()[0]
low_conf = cur.execute('SELECT COUNT(DISTINCT factor_code) FROM pit_factor_observations WHERE source_confidence <= 0.6').fetchone()[0]

print(f'总因子: {total} | 近7天: {recent7} | 低置信: {low_conf}')
conn.close()
```
