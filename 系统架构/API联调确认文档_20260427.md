## YIYI API 交付确认

### 1. IC 热力图 API

| 项目 | 确认内容 |
|------|----------|
| **端点** | `GET /api/ic/heatmap` |
| **IC 计算方法** | Spearman（非参数，对异常值稳健） |
| **品种筛选** | ✅ 支持，`symbols` 参数逗号分隔 |
| **更新频率** | 日终（收盘后计算） |
| **状态** | ✅ 已部署运行 |

**响应格式：**
```json
{
  "factors": ["momentum", "real_rate", "inventory_change", ...],
  "symbols": ["RB", "HC", "I", "J"],
  "icMatrix": [[0.0, 0.0, 0.0, 0.0], ...],
  "lookbackPeriod": 60,
  "holdPeriod": 5,
  "updatedAt": "2026-04-27T00:29:53Z"
}
```

**前端颜色映射：** 红 = 负 IC，绿 = 正 IC（符合中国市场习惯）

### 2. 信号系统 API

| 项目 | 确认内容 |
|------|----------|
| **端点** | `GET /api/signal/{symbol}` |
| **信号聚合** | IC 加权求和（权重 = IR 归一化） |
| **计算频率** | 日终（收盘后批量计算） |
| **状态** | ✅ 已部署运行 |

**信号强度定义：**

| 分值 | 信号 |
|------|------|
| 0-20 | STRONG_SELL |
| 20-40 | SELL |
| 40-60 | NEUTRAL |
| 60-80 | BUY |
| 80-100 | STRONG_BUY |
