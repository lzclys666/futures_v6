## 前端调用示例

### 使用 axios
```typescript
import axios from 'axios';

const api = axios.create({
  baseURL: '/api',
  timeout: 10000,
});

// 获取持仓
const positions = await api.get('/trading/positions');

// 下单
const order = await api.post('/trading/order', {
  symbol: 'RU2506',
  direction: 'long',
  offset: 'open',
  volume: 5,
  price: 15200.0,
  trade_type: 'limit'
});

// 撤单
await api.delete('/trading/order/RU2506-LONG-001');
```

## 错误码

| code | 说明 |
|------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 风控拦截 |
| 1003 | 资金不足 |
| 1004 | 持仓超限 |
| 2001 | 订单不存在 |
| 2002 | 撤单失败 |
| 5001 | 系统错误 |
