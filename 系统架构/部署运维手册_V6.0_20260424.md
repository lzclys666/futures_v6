## 6. 备份与恢复

### 6.1 备份策略
```powershell
# 每日备份脚本 backup_daily.ps1
$date = Get-Date -Format "yyyyMMdd"
$backupDir = "D:\backup\futures_v6\$date"

# 1. 备份配置
Copy-Item D:\futures_v6\config $backupDir\config -Recurse

# 2. 备份 CSV
Copy-Item D:\futures_v6\macro_engine\output $backupDir\output -Recurse

# 3. 备份 SQLite
Copy-Item D:\futures_v6\.vntrader\*.db $backupDir\db\

# 4. 备份策略
Copy-Item D:\futures_v6\strategies $backupDir\strategies -Recurse

# 5. 压缩
Compress-Archive $backupDir "$backupDir.zip"
```

### 6.2 恢复流程
```powershell
# 1. 停止所有服务
Stop-Process -Name python -Force
Stop-Process -Name node -Force

# 2. 恢复配置
Copy-Item $backupDir\config D:\futures_v6\config -Recurse -Force

# 3. 恢复数据
Copy-Item $backupDir\output D:\futures_v6\macro_engine\output -Recurse -Force
Copy-Item $backupDir\db\*.db D:\futures_v6\.vntrader\ -Force

# 4. 重启服务
.\start_production.ps1
```

## 7. 安全规范

### 7.1 访问控制
- CTP 密码：定期更换（90天）
- API 密钥：环境变量存储，不提交代码
- 前端：生产环境启用 HTTPS
- 数据库：SQLite 文件权限 600

### 7.2 数据安全
- CSV 输出：只读权限
- 日志文件：定期清理敏感信息
- 备份加密：7z 加密压缩

### 7.3 交易安全
- 模拟盘 → 小资金实盘 → 全资金实盘
- 单日最大亏损：5000元
- 总持仓上限：50手
- 熔断机制：宏观评分 < -0.15 暂停交易

## 9. 扩展指南

### 9.1 添加新品种
1. 创建配置文件：`config/instruments/NEW.yaml`
2. 创建爬虫：`crawlers/NEW/run_all.py`
3. 更新前端下拉框：`frontend/src/config/symbols.ts`
4. 重启服务

### 9.2 添加新因子
1. 实现因子类：`core/factors/new_factor.py`
2. 注册到 Pipeline：`core/pipeline/factor_node.py`
3. 更新权重配置：`config/weights/new_factor.yaml`
4. 回测验证

### 9.3 添加新风控规则
1. 实现规则类：`services/macro_risk_app.py`
2. 注册到规则引擎：`RiskRuleRegistry.register()`
3. 更新配置：`RISK_RULES['new_rule'] = threshold`
4. 单元测试：`scripts/test_new_rule.py`
